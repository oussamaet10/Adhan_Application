## description
L'application Adhan aide les utilisateurs à suivre les horaires de prière islamiques en fonction de leur localisation. Elle offre des notifications automatiques pour chaque prière, joue l'adhan en arrière-plan, et inclut une boussole pour indiquer la direction de la qibla.
